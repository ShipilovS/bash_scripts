4. Изучение процессов POSIX
Изучить теорию из присоединенного файла и написать на языке C программу prntxt для Linux, 
которая принимает в параметрах командной строки число повторов, величину таймаута и текст,
 а затем печатает на экран строку текста заданное количество раз с заданным таймаутом. 
Поддержать как короткие, так и длинные опции.
Синтаксис вызова
prntxt -n|--number <число повторов> [-t|--timeout [<таймаут>]] -- <произвольный текст>
Значение параметра <таймаут> сделать опциональным, по умолчаний 1 секунда. При 
вызове prntxt без параметра -t|--timeout 
выводить текст без таймаута.
При вызове prntxt без параметра -n|--number или если не указан текст для вывода, вывести 
сообщение об ошибке и подсказку по 
синтаксису (usage: prntxt -n|--number <N> [-t|--timeout [<T>]] -- <TEXT>)

5 Task
Изучить теорию из присоединенного файла и написать на языке C программу launch для Linux, которая
 принимает в первом параметре 
командной строки имя файла программы для запуска, а во втором параметре командной строки имя файла стандартного 
вывода для запущенной программы.
launch <prog> <file>
Программа launch должна запустить программу <prog> в дочернем процессе и перенаправить стандартный 
вывод запущенной программы в файл <file>, дождаться завершения дочернего процесса и вывести на экран 
PID и код возврата завершенного процесса.
